🚀 Next.js Full-Stack MERN Hub
Welcome to my full-stack development showcase. This repository contains two major applications built with Next.js, demonstrating end-to-end architecture, cloud integrations, and complex database management.

📂 Projects Overview
1. ☁️ Cloudinary SaaS (AI Video & Image Transformer)
A production-ready SaaS application that leverages Cloudinary's powerful API for real-time media manipulation.

Key Features:

AI Smart Crop: Automatically resize videos for social media platforms (Instagram, TikTok, LinkedIn).

Secure Uploads: Implementation of signed uploads and secure media handling.

Dynamic Image Optimization: Real-time filters and format conversions.

Tech Stack: Next.js, Prisma, PostgreSQL, Cloudinary API, DaisyUI/Tailwind CSS.

2. 💬 Stack Overflow Clone
A complex community-driven Q&A platform focused on high-performance data fetching and interactive user experiences.

Key Features:

Full CRUD: Create, read, update, and delete questions and answers.

Reputation System: Complex logic for upvoting and downvoting content.

Global Search: Optimized search functionality for navigating technical queries.

Tech Stack: Next.js, Appwrite (BaaS), Zustand (State Management), Shadcn/UI.
